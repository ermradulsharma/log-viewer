Once installed, you can access the LogViewer dashboard by visiting:

`http://yourdomain.com/log-viewer`

### Features Highlights:

- **Dashboard**: High-level overview of your log health.
- **Log List**: Browse all your log files.
- **Filtering**: Filter entries by level (Error, Critical, etc.).
- **Search**: Real-time search across entries or global search across files.
- **Team Notes**: Add persistent notes to specific log entries.
- **Saved Searches**: Quick access to frequently used filters.
- **AI Explain**: Get ML-powered insights for complex errors.

### Programmatic Usage

You can also use the `log_viewer()` helper or the `LogViewer` facade to interact with logs programmatically:

```php
// Get all logs
$logs = log_viewer()->all();

// Get a specific log
$log = log_viewer()->get('2023-01-01');

// Delete a log
log_viewer()->delete('2023-01-01');
```
